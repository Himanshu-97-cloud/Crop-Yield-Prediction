# 📁 Models Folder

This folder stores trained machine learning model files.

## 📌 What will be generated here

* best_model.pkl (trained model)
* scaler.pkl (feature scaler)
* label_encoders.pkl (encoders for categorical data)
* r2.txt (model performance score)

## ⚙️ How to generate these files

Run the following commands:

python src/preprocess.py
python src/train.py

## ⚠️ Note

These files are not included in the repository and will be created automatically when the above scripts are executed.
